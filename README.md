### 📝 General Rules & Project Format

1. **Case Study Selection:** Case studies will be revealed on **5/3/2026 10:00 AM (GMT +8)**. Each team can select only **one** case study from the AI domain.
2. **Platform Flexibility:** The solution can be a mobile application, desktop application, or website.
3. **Tech Stack:** Any suitable programming language is acceptable, but your team MUST develop a **working prototype**.
4. **Submission Portal:** The team leader must submit the project on behalf of the team via the [V HACK 2026 website](https://vhackusm.com/).


### 📦 Required Submission Materials

🐙 **1. Source Code Repository**

- Provide a GitHub link to your source code.
- ⚠️ **Crucial:** Ensure the repository is set to **'Public'** so our judges can access and evaluate your submission.

🎥 **2. Video Presentation**

- **Platform:** YouTube or Google Drive link.
    
- **Visibility:** Unlisted or Public.
    
- **Duration:** 5-10 minutes.
    
- **Naming Convention:** **`VHACK2026_[TEAM NAME]_[PROJECT TITLE]`**
    
    _E.g., **`VHACK2026_WeAreVHackers_VHACKProject`**_
    



### 📋 Video Content Checklist

Ensure your video presentation covers all of the following points:

- Team member introduction
- Chosen case study and reason(s)
- Proposed Solution
- Front-End Demo
- Suitable technologies used to build your solution
- Business model
- Market Segment
- Competitor Analysis
- Room for future improvements and potential expansion

> 🚨 **IMPORTANT:** **Please be strictly aware of the submission deadlines!**
